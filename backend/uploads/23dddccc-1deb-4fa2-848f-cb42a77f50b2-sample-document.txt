Abstract

This document presents an automated rule-based  hsa document processing system designed to standardise and format academic and professional documents. The system analyses uploaded documents and applies predefined formatting rules to ensure consistency and compliance with established standards.

Introduction

Document formatting and content standardisation are critical in academic, legal, and professional writing. Manual rule-checking and correction is time-consuming and error-prone. This project addresses these challenges by providing an automated solution.

The system accepts a Rules file (PDF/DOCX/TXT) containing formatting or editing instructions. It also accepts a Target document (PDF/DOCX/TXT) to be processed. The programme then applies the rules automatically.

Important notice: All users must review the output before final submission.

Methodology

The methodology involves several key steps:

bullet point First, the system parses the rules file to extract formatting instructions
bullet point Second, it analyses the target document structure
bullet point Third, it applies transformations based on the detected sections
bullet point Finally, it generates a formatted output with tracked changes
